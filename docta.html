Setup instructions (one-time)
1. Install Ollama: https://ollama.com/download 2. Start with CORS enabled: Windows: set OLLAMA_ORIGINS=* && ollama serve Mac/Linux: OLLAMA_ORIGINS="*" ollama serve 3. Pull a model (first time): ollama pull qwen2.5:7b 4. Click "check" above — dot turns green.
